# Research Skill

Basic web search functionality to gather context and news.

## Capabilities

### 1. Web Search

Finds the top 5 web results for a given query.

*   **Command:** `node skills/research/search.js "<query>"`

**Example:**
```bash
node skills/research/search.js "Latest updates on Ethereum upgrade"
```
